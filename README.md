## Modules

1. **Foundations** - Lab workflow, biosafety, media types, QC
2. **Gram Stains** - Pattern recognition and morphology
3. **Plates & Media** - Colony recognition and growth patterns
4. **Bacteriology** - ID pathways for major bacterial groups
5. **Mycobacteriology** - AFB stains, culture, and resistance testing
6. **Mycology** - Yeasts, molds, and dimorphic fungi
7. **Parasitology** - Ova & parasites, malaria, key parasites
8. **Virology** - PCR/NAAT, respiratory panels, viral diagnostics
9. **Molecular Diagnostics** - Multiplex panels and molecular stewardship
10. **AST** - MIC interpretation, breakpoints, resistance mechanisms
11. **Infection Control** - Isolation precautions, outbreak recognition, stewardship

## Interactive Tools

- **Plate Reader Simulator** - Practice colony identification
- **Gram Stain Interpreter** - Master morphological patterns
- **AST Analyzer** - Interpret MIC values and breakpoints
- **Contamination vs Infection Sorter** - Clinical correlation practice
